Exemple concret

Bug : compte admin inaccessible

1
Humain

Signale le bug : "Invalid credentials" sur admin@verticallog.fr / admin123

2
IA

Explore les fichiers auth. Identifie que le hash bcrypt dans init.sql est invalide pour le mot de passe admin123

3
IA

Génère un nouveau hash via Node.js, corrige init.sql, met a jour la base de données directement

4
Résultat

bug resolu en ~2 minutes

Evaluation critique

Bilan quantitatif & qualitatif

Points forts

  • Productivité élevée sur les tâches répetitives (CRUD, types TypeScript)
  • CLAUDE.md évite de ré-expliquer l'architecture à chaque session
  • Gestion git intégrée : commits clairs
  • Explique ses choix techniques en temps reel

Limites

  • Ne peut pas tester visuellement l'interface utilisateur
  • Génère parfois plus de code que nécessaire — supervision requise
  • Contexte perdu si le CLAUDE.md est absent ou incomplet
  • La décision finale reste toujours côté humain
  • Nécessite de savoir ce qu'on veut pour être efficace
